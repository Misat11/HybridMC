# HybridMC
HybridMC is plugin that allows connection of Bedrock Edition clients on your Java Edition Spigot or Craftbukkit server!

## Features
- [ ] Connection to downstream server
- [ ] Loading map
- [ ] Moving
- [ ] Spawning entities
- [ ] Inventory support
- [ ] Sounds
- [ ] Effects
- [ ] API for forms
- [ ] Skins
- [ ] Multiversion
